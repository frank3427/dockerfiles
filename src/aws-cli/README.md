## Supported tags and respective Dockerfile links

* `latest` [(Dockerfile)](https://github.com/qmu/dockerfiles/blob/master/src/aws-cli/Dockerfile)

## Usage

```
docker run --rm qmu1/aws-cli:latest --version
```
